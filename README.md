
# Authing 

Simple Authentication and Authorization using Auth0

## Installation

Follow the steps in auth0.com/docs to make a account in Auth0

```bash
  git clone https://github.com/sDaCoder/authing.git .
  pnpm install
  pnpm run dev
```
